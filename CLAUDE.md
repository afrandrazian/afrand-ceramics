# Afrand Ceramics

Handmade ceramics shop. Two storefronts (Etsy + personal website), one inventory, auto-synced. The website gives the shop owner brand control and lower fees (Stripe 2.9% vs Etsy ~10-12%).

For full technical context, platform comparisons, and fee breakdowns: see `docs/context.md`

## Who Works Here

- **Shop owner** — content, branding, design, products, business decisions
- **Cowboy developer** — sync scripts, deploy pipeline, automation

Both use Claude Code. Everything is git-tracked — experiment freely.

## How It Deploys

`main` branch = production (once Cloudflare Pages is connected).
Push to main → auto-deploys to afrandceramics.com.
Push to any other branch → Cloudflare generates a preview URL.

Use branches + PRs for changes you want reviewed before going live.
Direct pushes to main go live immediately.

## How the Sync Works

**GitHub Actions** (`.github/workflows/sync-etsy.yml`):
- Pulls listings from Etsy API → writes `products.json` → commits as "Etsy Sync Bot"
- Runs every 30 min (currently disabled — Etsy API not yet approved)
- Auto-rotates OAuth refresh tokens (Etsy tokens are single-use)
- `products.json` is auto-generated — manual edits get overwritten on next sync

**overrides.json**:
- Merges on top of `products.json` at render time
- Key = Etsy listing ID (string). Example: `{ "12345": { "hide": true } }`
- Never overwritten by the bot — safe to edit anytime

**Secrets needed** (configure after API approval):
`ETSY_API_KEY`, `ETSY_REFRESH_TOKEN`, `ETSY_SHOP_ID`, `GH_PAT`

## Project Structure

```
index.html          — the entire website (single file, CSS inline, vanilla JS)
products.json       — product catalog (auto-generated by sync bot)
overrides.json      — manual product tweaks (hide items, override display)
sync/               — Etsy API sync scripts (zero npm dependencies)
setup/              — one-time OAuth setup script
docs/               — project docs + handoff materials
  brief.html        — visual project overview (open in browser)
  context.md        — full technical reference
  etsy-api-application.md — Etsy API form values (copy-paste ready)
.github/workflows/  — GitHub Actions automation
```

## Design Tokens

- Backgrounds: `--pampas: #f1ede8`, `--wood: #f4f0e9`, `--ebb: #efeaea`
- Accent: `#b07a55` (clay — used inline, not a variable)
- Text: `#2d2318`, muted: `#6b5c4e`, faint: `#9e9087`
- Grid: 4-col → 2-col at 1024px → 1-col at 600px
